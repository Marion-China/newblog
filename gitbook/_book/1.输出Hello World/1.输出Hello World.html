怎么样？

一.书写基本框架

首先头文件：

#include <iostream>

C++库（方便编辑）： 

using namespace std;

基本框架：

int main()
{
}

结束：

return 0; 然后就基本完成了：

#include <iostream>
using namespace std;
int main()
{
    return 0;
}

这样的话什么都不会输出

二.输出内容

cout<<"你要说的话";

然后：

#include <iostream>
using namespace std;
int main()
{
    cout<<"你要说的话";
    return 0;
}

三.换行的方式

假设你想书写一段话：你可能尝试这么写：

#include <iostream>
using namespace std;
int main()
{
    cout<<"Hello World";
    cout<<"我要换行！";
    return 0;
}

但是可能运行后：

法一：加<<endl;

在文本双引号后输入<<endl; 如下：

#include <iostream>
using namespace std;
int main()
{
    cout<<"Hello World"<<endl;
    cout<<"我要换行！";
    return 0;
}

这样：

法2:加\n

在文本内输入\n

#include <iostream>
using namespace std;
int main()
{
    cout<<"Hello World\n";
    cout<<"我要换行！";
    return 0;
}

效果相同，但是你可以这样：

例题3：输出图形

#include <iostream>
using namespace std;
int main()
{
cout<<" ##     ##    ##"<<endl;
cout<<"  ############"<<endl;
cout<<"    ########"<<endl;
cout<<"     ######"<<endl;
cout<<"       # "<<endl;
    return 0;
}

例题四：输出长方形

#include <iostream>
using namespace std;
int main()
{

cout<<"####################"<<endl;
cout<<"#                  #";
cout<<"#                  #";
cout<<"#                  #";
cout<<"#                  #";
cout<<"####################";
    return 0;
}
